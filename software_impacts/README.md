# Software Impacts submission package (DaCaF)

Working files for the **Original Software Publication (OSP)** companion paper to the
*Information Fusion* (2026) article (DOI `10.1016/j.inffus.2026.104517`), submitted to
Elsevier **Software Impacts**.

## Contents

| File | Purpose |
|---|---|
| `CHECKLIST.md` | Full Phase 0-4 submission plan, NIH-data note, leverage items. |
| `dacaf_software_impacts.tex` | The metapaper. Bullet points are placeholders to be replaced with prose. Built on the official CAS class. |
| `els-cas/` | Official Elsevier CAS class files (v2.4) from `els-cas-templates.zip`: `cas-dc.cls` (double column), `cas-sc.cls` (single column), `cas-common.sty`, `cas-model2-names.bst`. |
| `chd49_crosstab_regenerated.csv` | Clean CHD-49 cross-tab regenerated from the shipped code (PCC + LR, 5 seeds x 10-fold); source for Table 1 in the metapaper. |

## Build

From this directory (the class files live in `els-cas/`, so put it on `TEXINPUTS`):

```bash
TEXINPUTS=".//:els-cas//:" pdflatex -interaction=nonstopmode dacaf_software_impacts.tex
TEXINPUTS=".//:els-cas//:" pdflatex -interaction=nonstopmode dacaf_software_impacts.tex   # 2nd pass for refs
```

LaTeX packages required by the CAS class (install via `tlmgr` if missing):
`makecell`, `moreverb`, `upquote`, `xstring`, `footmisc`, `xspace`.

## Note on generative AI

AI tooling (Claude Code) was used to scaffold this submission package (checklist,
LaTeX skeleton, regenerated CHD-49 table). Per the Software Impacts Guide for Authors,
this must be disclosed in the manuscript's "Declaration of generative AI" section before
the references. The numbers in `chd49_crosstab_regenerated.csv` were produced by running
the repository's own code, not generated by an AI tool.
