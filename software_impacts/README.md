# Software Impacts submission package (DaCaF)

Files for the **Original Software Publication (OSP)** companion paper to the
*Information Fusion* (2026) article (DOI `10.1016/j.inffus.2026.104517`), submitted to
Elsevier **Software Impacts**.

## Contents

| File | Purpose |
|---|---|
| `dacaf_software_impacts.tex` | The OSP manuscript, written on the official Software Impacts article template (Version 2, July 2021). |
| `dacaf_software_impacts.pdf` | Compiled manuscript. |
| `highlights.txt` | Highlights (3-5 bullets, <=85 chars each), submitted as a separate file. |
| `chd49_crosstab_regenerated.csv` | CHD-49 cross-tab regenerated from the shipped code (PCC + LR, 5 seeds x 10-fold); source for Table 1. |

## Build

The manuscript uses the standard `article` class, so a plain build works:

```bash
pdflatex dacaf_software_impacts.tex
pdflatex dacaf_software_impacts.tex   # 2nd pass for references
```

## Note on generative AI

AI tooling (Claude Code) was used while preparing this submission package. Per the
Software Impacts Guide for Authors, if AI tools were used in writing they must be
disclosed in the manuscript's "Declaration of generative AI" section. The numbers in
`chd49_crosstab_regenerated.csv` were produced by running the repository's own code,
not generated by an AI tool.
